# Карта Дня - План проекта

## Описание
Интерактивная веб-страница с картой дня. Пользователь выбирает категорию, нажимает на карту, она переворачивается и показывает мотивирующее послание.

## Текущий статус: v2.0 (Полная версия)

## Реализованные функции

## Для разработчиков

### Добавление новых пожеланий
В `index.html` найди объект `wishesData` и добавь в нужную категорию:
```javascript
wishesData.love.push("Новое пожелание о любви!");
```

### Добавление новой категории
1. Добавь массив пожеланий в `wishesData`
2. Добавь иконку в `categoryIcons`
3. Добавь название в `categoryNames`
4. Добавь кнопку в HTML (div.categories)

### Создание иконок PWA
Нужны PNG-иконки размером 192x192 и 512x512 пикселей.
Положить в папку `imgs/` как `icon-192.png` и `icon-512.png`.
