# Карта Дня - План проекта

## Описание
Интерактивная веб-страница с картой дня. Пользователь нажимает на карту или кнопку, карта переворачивается и показывает случайное мотивирующее послание.

## Текущий статус: MVP готов

## Структура проекта
```
mini-crm-savings/
├── index.html          # Главная страница (всё в одном файле)
├── imgs/
│   └── card-back.png   # Рубашка карты (дизайн)
├── PROJECT_PLAN.md     # Этот файл
└── README.md           # Описание проекта
```

## Функции (реализовано)
- [x] Отображение карты рубашкой вверх
- [x] Кнопка "Взять карту дня"
- [x] Анимация переворота карты (3D flip)
- [x] 36 мотивирующих пожеланий
- [x] Случайный выбор пожелания
- [x] Возможность взять другую карту
- [x] Звёздный фон с анимацией
- [x] Эффект искр при перевороте
- [x] Адаптивный дизайн (мобильные устройства)

## Для продолжения разработки

### Добавление новых пожеланий
В файле `index.html` найди массив `wishes` и добавь новые строки:
```javascript
const wishes = [
    "Существующее пожелание...",
    "Новое пожелание здесь!"
];
```

### Изменение дизайна рубашки
Замени файл `imgs/card-back.png` на новое изображение.

### Изменение стилей
Все стили находятся в теге `<style>` внутри `index.html`.
