# Карта Дня

Интерактивная веб-страница с ежедневными мотивирующими посланиями в формате карт.

## Демо
Нажмите на карту или кнопку "Взять карту дня" чтобы получить своё послание!

## Функции
- Красивая анимация переворота карты
- 36+ мотивирующих пожеланий
- Магический дизайн с эффектами
- Адаптивный интерфейс

## Запуск
Просто откройте `index.html` в браузере.
